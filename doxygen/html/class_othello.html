Detailed Description

Extends Game with the rules of Othello as specified.

\file -Othello.Java

Author
-B. Golightly, Mathew Lloyd 711293
Date
-12/02/2014

Methods are move so that the AI can know how good it is, and provides a way to take a move, return a list of changes after a move has been made. In the case of Othello this means returning a list of captured pieces; other classes that also extend game may only return a list of one item i.e. only the piece that is the result of the move.
